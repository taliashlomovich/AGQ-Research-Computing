# AGQ Computing Course – Jupyter Notebooks

This repository contains my completed Jupyter notebooks for the AGQ Computing Course. 

The notebooks cover a range of computational and machine learning techniques,
including neural networks, filtering methods, variational autoencoders, and
classification problems in scientific data.

## Contents

The repository includes notebooks on:

- DNN – experiments with deep neural networks using NumPy and PyTorch 
- Filters and VAE – filtering methods and variational autoencoders
- ATLAS – data analysis inspired by particle physics datasets
- Knot Classification – machine learning applied to knot classification problems
